## 🚀 快速开始

### 环境要求

- Node.js >= 16
- npm >= 7

### 安装

```bash
# 克隆项目
git clone https://github.com/aidevelopcn/aiweb.git

# 进入项目目录
cd your-project

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建

```bash
# 构建生产环境
npm run build
```

## 📦 项目结构

```
├── public/                 # 静态资源
├── src/                    # 源代码
│   ├── assets/            # 项目资源
│   ├── components/        # 公共组件
│   ├── layouts/           # 布局组件
│   ├── router/            # 路由配置
│   ├── store/             # 状态管理
│   ├── styles/            # 全局样式
│   ├── utils/             # 工具函数
│   ├── views/             # 页面组件
│   ├── App.vue            # 根组件
│   └── main.js            # 入口文件
├── .gitignore             # Git 忽略文件
├── index.html             # HTML 模板
├── package.json           # 项目配置
├── vite.config.js         # Vite 配置
└── README.md              # 项目说明
```

## 🔧 配置说明

项目的主要配置文件位于 `vite.config.js`，你可以根据需要修改以下配置：

- 开发服务器配置
- 构建配置
- 路径别名
- 插件配置
